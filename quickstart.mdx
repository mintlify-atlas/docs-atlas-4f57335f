## Get started with Barro Tal Vez

Get the pottery workshop management system running on your local machine with Docker Compose.

<Steps>
  <Step title="Prerequisites">
    Make sure you have the following installed:
    - Docker Desktop (v20.10 or higher)
    - Git
    
    <Note>
      Docker Desktop includes both Docker and Docker Compose, which are required to run Barro Tal Vez.
    </Note>
  </Step>

  <Step title="Clone the repository">
    Clone the Barro Tal Vez repository to your local machine:
    
    ```bash
    git clone https://github.com/massimoquirolo/barro-tal-vez-v2.git
    cd barro-tal-vez-v2
    ```
  </Step>

  <Step title="Start the application">
    Use Docker Compose to build and start all services:
    
    ```bash
    docker-compose up --build
    ```
    
    This command will:
    - Build the frontend (React + Vite)
    - Build the backend (Express.js + Prisma)
    - Start PostgreSQL database
    - Set up networking between services
    
    <Note>
      The first build may take 5-10 minutes. Subsequent starts will be much faster.
    </Note>
  </Step>

  <Step title="Initialize the database">
    In a new terminal window, run the Prisma migrations and seed script:
    
    ```bash
    # Access the backend container
    docker exec -it barro-api sh
    
    # Run Prisma migrations
    npx prisma migrate deploy
    
    # Seed initial data
    node prisma/seed.js
    
    # Exit the container
    exit
    ```
    
    This creates a test user with the following credentials:
    - **Email**: test@barro.com
    - **Password**: 123
  </Step>

  <Step title="Access the application">
    The application is now running at:
    
    - **Frontend**: [http://localhost:8082](http://localhost:8082)
    - **Backend API**: [http://localhost:3001](http://localhost:3001)
    - **Database**: localhost:5433
    
    <CardGroup cols={2}>
      <Card title="Student Dashboard" icon="user">
        Login with test credentials to access the student view and manage your pottery pieces.
      </Card>
      <Card title="Admin Dashboard" icon="shield">
        Create an admin user to access inventory, billing, and student management features.
      </Card>
    </CardGroup>
  </Step>

  <Step title="First login">
    1. Open [http://localhost:8082](http://localhost:8082) in your browser
    2. Login with:
       - Email: `test@barro.com`
       - Password: `123`
    3. You'll be prompted to change your password on first login
    
    <Warning>
      The default credentials are for development only. Always use strong passwords in production.
    </Warning>
  </Step>
</Steps>

## Verify the installation

Confirm everything is working correctly:

```bash
# Check running containers
docker ps

# You should see three containers:
# - barro-web (frontend)
# - barro-api (backend)
# - barro-db (database)
```

## Stop the application

When you're done, stop all services:

```bash
docker-compose down
```

To remove volumes (database data):

```bash
docker-compose down -v
```
